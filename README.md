Project Overview
PrismPlan is an AI-powered system designed to assist architects and interior designers during the early stages of floor plan creation. By leveraging Graph Neural Networks, similarity retrieval techniques, and generative AI models, the platform can retrieve existing layouts and generate new floor plan proposals from incomplete user specifications.
The project is developed in collaboration with Maticad and explores how AI can be integrated into professional CAD workflows to accelerate design exploration and decision-making.
